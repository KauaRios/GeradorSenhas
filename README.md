# 🔐 Gerador de Senhas Seguras (CLI)

🚧 **Projeto em desenvolvimento** 🚧

Um gerador de senhas seguras em linha de comando (CLI), que permite ao usuário criar senhas fortes e aleatórias com base em preferências personalizadas.

---

## 📌 Sobre o Projeto

Este programa foi desenvolvido com o objetivo de praticar lógica de programação e conceitos básicos de segurança.

O usuário pode definir:

* O tamanho da senha
* Se deseja incluir símbolos especiais

A partir dessas escolhas, o sistema gera uma senha aleatória utilizando diferentes tipos de caracteres.

---

## 🚀 Funcionalidades

* Geração de senhas aleatórias
* Escolha do tamanho da senha
* Inclusão opcional de símbolos especiais
* Execução via terminal

---

## 🧠 Conceitos Praticados

* Manipulação de strings (`std::string`)
* Uso de arrays/listas
* Estruturas de repetição (`for`)
* Geração de números aleatórios
* Entrada e saída de dados

---

## 🛠️ Tecnologias Utilizadas

* C++

## 📥 Como Executar

### 1. Clone o repositório

git clone https://github.com/seu-usuario/gerador-senhas-cli.git

### 2. Acesse a pasta

cd gerador-senhas-cli

### 3. Compile o programa

g++ main.cpp -o gerador

### 4. Execute

./gerador

---

## 💻 Exemplo de Uso

Qual o tamanho da senha? 10
Deseja incluir símbolos? (s/n): s

Senha gerada: K8#fL2!xP@

## 🔒 Boas Práticas
